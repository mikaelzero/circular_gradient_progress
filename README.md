# circular_gradient_progress

## Usage

```dart
 CircularGradientCombineWidget(
   size: 120,
   duration: duration,
   centerCircleSizeRatio: 0.3,
   sweepAngles: const [600, 720, 860],
   backgroundColors: backgroundColors,
   gradientColors: gradientColors,
 ),
```
